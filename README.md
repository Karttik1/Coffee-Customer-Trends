# ☕ Coffee Customer Trends Dashboard

A Power BI dashboard analyzing coffee sales trends across US, UK, and other regions.

## 📊 Dataset Source
- [Coffee Bean Sales Dataset - Kaggle](https://www.kaggle.com/datasets/saadharoon27/coffee-bean-sales-raw-dataset)
- [Coffee Chain Sales Dataset - Kaggle](https://www.kaggle.com/datasets/amruthayenikonda/coffee-chain-sales-dataset)

## 📁 Files
- `Coffee Consumer Trends.pbix` — Main Power BI Dashboard file

## 🔍 Key Insights
- Sales by Country (US, UK, Ireland)
- Coffee Type wise breakdown (Arabica, Robusta, Excelsa, Liberica)
- Regional trends and top customers

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Kaggle Datasets
